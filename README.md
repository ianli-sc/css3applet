# css3applet
==========

## What's this?
* Use css3 to create appearance of checkbox, input, select, radio...to instead of the default for Mobile browser which able to get `same good looking`~
* Supported:
  * `android 4.x`
  * `android 2.3.x`
  * `ios 6.x`
  * `ios 5.x`
  * `wp 8.0.x`
